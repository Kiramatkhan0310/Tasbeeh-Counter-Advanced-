# Tasbeeh-Counter-Advanced-
📿 Islamic Tasbih Counter
    A modern and interactive Tasbih Counter App built using HTML, CSS, and JavaScript. This
    application helps users perform dhikr digitally with Islamic phrases, sound effects,
    responsive design, and dynamic image updates.


    🚀 Features
    📿 Digital Tasbih Counter
    🔊 Islamic sound effects for each dhikr
    🖼️ Dynamic image updates based on count
    🔄 Automatic reset after 100 counts
    🎨 Modern Islamic-themed UI
    📱 Fully responsive design
    🔇 Sound mute/unmute option
    🛠️ Technologies Used

        HTML5
        CSS3
        JavaScript (Vanilla JS)

    🎯 Learning Outcomes
    DOM manipulation
    Event handling
    Audio playback in JavaScript
    Dynamic image changing
    Responsive UI design
    Counter logic and conditional statements
